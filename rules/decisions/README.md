# Decisions

This directory contains decision records that explain why current rules exist.

Decision records are not a substitute for the active rulebooks:

- Active draft rules live in `rules/draft/`.
- Active review rules live in `rules/review/`.
- Decision history and rationale live here.

Agents may read this directory when the active rulebooks do not fully resolve an
ambiguous case. Agents must not treat an unapproved idea in a decision record as
a new active rule unless the active rulebook or user confirms it.
