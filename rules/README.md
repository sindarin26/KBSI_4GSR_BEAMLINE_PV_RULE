# Rules

Rulebooks live here.

- `draft/`: rules for generating PV draft outputs.
- `review/`: rules for validating generated PV data and documents.
- `decisions/`: decision records explaining why rules exist and how ambiguous
  cases were judged.

Active PV naming rules should be promoted into `draft/` and `review/`.
Decision records in `decisions/` explain context, but they are not a substitute
for concise active rulebooks.
