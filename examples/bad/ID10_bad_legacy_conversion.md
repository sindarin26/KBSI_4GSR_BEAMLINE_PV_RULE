# Bad Example: Blind Legacy Conversion

These examples should be rejected by the v0 review rulebook.

```text
ID10:IVU:Gap
ID10:PTL:IVU:GirderY
ID10:OH:WBS:Top
ID10:OH:WBSLIT01:hgap
ID10:OH:PH1:m1
```

Problems:

- missing area tier in `ID10:IVU:Gap`;
- CamelCase legacy suffixes;
- legacy WBS/WBSLIT instead of `WBSLT`;
- one-digit numbering.
